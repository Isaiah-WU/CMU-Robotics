## Topics Covered

- **Foundations** — ML & deep learning refreshers for robotics
- **Imitation Learning** — behavior cloning, DAgger, learning from demonstrations
- **Reinforcement Learning** — MDPs, value-based methods (Q-Learning), policy-based methods (Policy Gradient, Actor-Critic), and advanced algorithms
- **Model-Based Methods** — model-based RL and optimal control (LQR, MPC)
